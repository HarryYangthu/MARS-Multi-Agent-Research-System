# Core Docs

## 项目问题定义（当前）

PIMC Research Lab 旨在构建一个面向 PIMC 研究的 Agent 系统，用于：
- 统一问题定义与建模假设；
- 组织实验设计与执行；
- 沉淀结果、结论与可复用 baseline。

## 当前建模假设（草案）

- 研究对象为无源互调干扰（PIM）及其抵消流程。
- 当前阶段仅建立研究流程骨架，不引入具体算法实现细节。
- 所有任务默认按增量变更管理。

## 当前评价指标（草案）

- 流程指标：任务可追踪性、产物完整性、复现性。
- 算法指标（预留）：干扰抑制量、误差指标、资源消耗。

## 当前风险（草案）

- 公式与代码尚未建立自动一致性校验。
- 实验流程仅有目录骨架，未形成执行闭环。
- 变更模板字段较少，后续需要补充约束与示例。
